# Role: Apple-inspired Web App Developer
# Goal: 이미지를 분석해 정교한 학교 '기안문' 초안을 생성하는 미니멀 웹 앱 제작

### 1. Visual Vibe (Apple Style)
- **UI Concept:** Apple 공식 홈페이지와 같은 극강의 심플함과 깨끗함.
- **Design Elements:** - 배경은 완전한 화이트(#FFFFFF) 또는 아주 연한 그레이(#F5F5F7).
  - 폰트는 'Pretendard' 또는 'SF Pro' 스타일의 산세리프체 사용.
  - 여백(Padding)을 아주 넉넉하게 사용하여 시각적 피로도 최소화.
  - 모든 버튼은 부드러운 라운드 처리와 미세한 그림자(Subtle Shadow).
  - 로딩 애니메이션은 부드러운 페이드인 효과 적용.

### 2. Core Function (Image to Draft)
- **Input:** 사용자가 기안 내용이 적힌 사진(메모, 화이트보드, 공문 캡처 등)을 업로드하거나 드래그 앤 드롭.
- **Processing:** - AI(Vision API)가 이미지 속 텍스트를 분석.
  - 학교 기안문 형식(제목, 근거, 목적, 일시, 장소, 대상, 세부내용, 행정사항)에 맞게 텍스트를 재구성.
- **Output:** 오른쪽에 실제 기안문 양식과 유사한 깔끔한 프리뷰 화면을 보여줌.

### 3. Key Interaction
- 사용자가 이미지를 올리면 "기안문을 정제하고 있습니다..."라는 세련된 문구 노출.
- 결과물은 '텍스트 복사하기' 버튼 하나로 간단히 가져갈 수 있게 함.
- 반응형 레이아웃: 모바일에서도 기안 내용을 확인할 수 있어야 함.

"자, 이제 이 비전을 바탕으로 첫 번째 프로토타입 코드를 작성해줘."
